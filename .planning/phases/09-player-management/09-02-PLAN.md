## must_haves

- The Online tab displays a live list of players sourced from `playerStore`.
- Each online player row is tappable and opens a native action sheet with Kick, Ban, Op/Deop, Set Gamemode, and Cancel.
- Kick and Ban actions prompt for an optional reason via `PromptModal` before dispatching the command.
- Gamemode selection is restricted to the four valid vanilla modes and is dispatched via `usePlayerActions`.
- `usePlayerActions` serializes commands through `ServerProcess.sendCommand` and exposes a global `isLoading` flag.
