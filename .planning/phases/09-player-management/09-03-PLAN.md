- Banned-ips tab allows entering an optional reason and passes it to `modifyPlayerList`.
- `modifyPlayerList` accepts an optional `reason` parameter and includes it in `ban` / `ban-ip` console commands.
- Operators tab displays each op with a `Badge` showing their level and a "Demote" button that removes them from `ops.json`.
